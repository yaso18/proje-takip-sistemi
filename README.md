# 🚀 Proje Takip Sistemi (Project Tracking System)

Bu proje, geliştirme süreçlerini daha düzenli yönetmek, görevleri takip etmek ve projelerin ilerlemesini görselleştirmek amacıyla geliştirilmiş bir **Flask** uygulamasıdır.

## 🛠️ Kullanılan Teknolojiler

| Teknoloji | Kullanım Amacı |
| :--- | :--- |
| **Python / Flask** | Backend ve Sunucu Yönetimi |
| **PostgreSQL** | Veritabanı ve Veri Saklama |
| **HTML5 / CSS3** | Arayüz ve Tasarım |
| **JavaScript** | Dinamik Etkileşimler |

## ✨ Temel Özellikler

- 📋 **Proje Yönetimi:** Yeni projeler oluşturun, düzenleyin ve silin.
- ✅ **Görev Takibi:** Projelere ait alt görevler (tasks) ekleyin.
- 📊 **Durum Güncelleme:** Projeleri "Devam Ediyor", "Tamamlandı" veya "Beklemede" olarak işaretleyin.
- 🔍 **Filtreleme:** Projeler arasında hızlıca arama yapın.
